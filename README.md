剑指offer 52 求两个链表的第一个公共节点
这题有五种方法：
1. 蛮力法
2. 辅助栈
3. 长度法，分别测量两个链表的长度，谁长谁先走他们之间长度差的步数，然后再一起走进行比对
		时间复杂度O(m+n),空间复杂度O(1)
4. 构建法，构建为一个带有环的链表，然后按照环的处理方式进行
5. 相遇法，L1+C, L2+C,首先让curA走L1+C步，curA再走L2步，总共L1+C+L2
           同理，让curB走L2+C步，curB再走L1步，总共也是L2+C+L1,
           那他们肯定相遇*/
		时间复杂度O(m+n),空间复杂度O(1)
